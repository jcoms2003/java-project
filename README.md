#### Instructions
- To use you will need to download and save the code files you your computer. You will also need an upto date java JDK package to run on your computer.
Once everything is downloaded you will execute the following into your CLI (I am using bash for Linux):
```bash
javac FILENAME.java
java FILENAME.java
```
- After execution of the bash command you will be prompted through the game. To exit, just hit ctr + q or ctr + c at any time.
